# 第 10 周：PWM 驱动舵机与直流电机

## 学完后能解释

舵机和直流电机都能接收 PWM，但它们需要的是两种不同信号。常见舵机每约 20 ms 接收一次脉冲，本实验只接受 500–2500 µs：脉宽决定目标角度，而不是用高频 PWM 的平均电压直接调速。TIM2 用 1 MHz 计数、`ARR=19999` 形成 50 Hz，CCR1 的数值就等于微秒数。直流电机则通过 TB6612 H 桥改变电流方向，TIM3 以 20 kHz PWM 调节平均功率；`AIN1/AIN2` 决定方向，`PWMA` 决定速度，`STBY` 必须拉高才允许驱动。

电机绕组是感性负载，切换电流时会储能和释放能量，启动/堵转电流也远高于 GPIO 能力。TB6612 提供功率开关和续流路径，但不能消除错误接线风险。舵机与电机必须独立供电，负载电源地、驱动板地和 STM32 地必须共地，这样 3.3 V 控制信号才有共同参考。GPIO 只能输出控制信号，绝不直接给负载供电。SPL 中 `TIM_OC1Init`、`TIM_OCInitTypeDef`、`TIM_OCMode`、`TIM_OCPolarity` 等概念，在 HAL 主线中由 CubeMX 配置和 `HAL_TIM_PWM_ConfigChannel`/`HAL_TIM_PWM_Start` 承担。

## 学完后能做到

能用 TIM2_CH1/PA0 输出 50 Hz 舵机脉冲，用 TIM3_CH1/PA6 输出 20 kHz 电机 PWM，并用 PA1、PA2、PB0 控制 TB6612 的方向与 STBY。能在 `App/actuators.c` 中先验证命令，再触碰 CCR 或 GPIO：舵机拒绝小于 500 或大于 2500 µs，电机拒绝小于 -1000 或大于 1000 的命令。能解释为什么自动构建和寄存器值只能证明软件边界，运动、声音和电流只能人工验证。

## 概念模型

把舵机命令理解成“每 20 ms 发一次长度明确的目标纸条”；把电机控制理解成“方向开关加快速功率开关”。两个频率差异巨大，不能共用同一个 timer period，因此使用 TIM2 和 TIM3。边界保护像功率系统的门卫：任何越界值在写比较寄存器、方向脚或 STBY 之前就返回错误。安全接线是另一道独立门卫，软件检查无法发现没有共地、负载电源过流或旋转部件卡住。

## CubeMX 为什么这样配

TIM2 使用 72 MHz 定时器时钟，PSC=71、ARR=19999，PA0 为 TIM2_CH1 PWM；TIM3 同样 PSC=71，但 ARR=49，PA6 为 TIM3_CH1 PWM，得到 20 kHz。PA1/PA2 配成低速推挽输出连接 AIN1/AIN2，PB0 配成推挽输出连接 STBY，初始都为低，保证上电默认不驱动。两个 PWM 初始 pulse 为 0，由 `Actuators_Init` 启动后再设置安全值。所有接线在断电状态完成，逻辑侧保持 3.3 V；舵机和电机使用独立供电并与 STM32 共地。

## 最小实验

1. 断电后连接 PA0→舵机信号、PA6→PWMA、PA1/PA2→AIN1/AIN2、PB0→STBY；检查 TB6612 VM 和负载电源极性。
2. 舵机、电机使用满足额定电压和电流的独立供电，把所有地可靠共地，再上电。
3. 先发送舵机 1500 µs 和电机 0，确认无异常发热；随后测试舵机 1000/1500/2000 µs 与电机 -600/0/600。
4. 调用 499、2501、-1001、1001 等越界命令，确认函数返回 `HAL_ERROR` 且原 CCR/方向输出不改变。
5. 人工记录角度、方向、声音和电流。ELF、静态检查或 CCR 数值均不能宣称负载已经运动。

## 调试与寄存器观察

先检查 `TIM2->PSC/ARR/CCR1` 是否代表 20 ms 周期与 500–2500 µs，再检查 `TIM3->PSC/ARR/CCR1` 是否代表 20 kHz 与 0–1000‰。若舵机抖动，查看负载电源压降、地线和脉宽；若电机不动，按 `STBY→AIN1/AIN2→PWMA→VM→电机→GND` 路径排查。寄存器正确而负载不动时，不要继续提高占空比，应先断电检查共地、STBY 和独立供电容量。

## 故障注入

在断电状态保持 STBY 低，或临时断开控制地但保留负载断电，先预测控制信号为什么失去有效参考。恢复共地、确认极性后再上电测试。软件故障注入使用越界命令，观察函数必须在 `__HAL_TIM_SET_COMPARE` 和 `HAL_GPIO_WritePin` 前拒绝。不要用拔插带电电机线的方式制造故障，也不要绕过 TB6612 让 GPIO 直接接电机。

## 复述检查

为什么舵机 50 Hz 与电机 20 kHz 需要两个定时器？500–2500 µs 如何映射到 1 MHz 计数器？电机命令 -600 的符号和绝对值各控制什么？STBY 低、没有共地、独立供电不足分别可能看到什么症状？解释为什么 CCR 正确仍不能证明舵机角度、电机声音或实际电流正确。
