# 二进制与 C 语言先修补救

## 目标

预计时间：30 分钟。能用位运算读写一个标志位，并用条件和循环描述等待外设状态的最小程序。

## 解释

寄存器由多个 bit 共享，一个掩码只选择相关位。`&` 用于检查，`|` 用于置位，`~mask` 配合 `&` 用于清位；循环必须有退出条件或超时。

## 微型练习

给定数值 `0b00101000`，写出检查 bit 3、置位 bit 6、清除 bit 5 的表达式，再写一个最多循环 100 次的等待伪代码。

## 检查问题

为什么直接把整个寄存器赋值为一个掩码可能破坏其他功能，而读改写可以降低这个风险？

## 通过证据

提交三个位运算结果和带上限的等待伪代码，并逐步列出每次运算前后的二进制值。
