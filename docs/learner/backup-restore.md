# 网页进度备份与恢复

网页把当前周、测验证据、阶段结果、补修队列和笔记保存在这台浏览器的 IndexedDB 中。清理浏览器数据或换电脑会丢失本地副本，所以更新前先导出。

## 导出

1. 打开“笔记与备份”。
2. 先保存正在编辑的“本周笔记”。
3. 点“导出全部进度”，保存下载的 JSON 文件；文件是文字数据，不是可执行程序。
4. 用日期改名并复制到另一个位置，例如云盘或移动硬盘。

## 导入与确认

1. 在“导入备份”选择 JSON 文件。
2. 网页会先验证版本与字段，并显示摘要；这一步还不会覆盖当前数据。
3. 核对当前周、证据数量和时间，再点“恢复已选备份”。
4. 恢复前网页会为当前状态做快照。恢复完成后检查首页当前周、笔记和知识报告。

如果提示无效，不要手改到“能导入”为止；保留原文件，按 [故障排查](troubleshooting.md) 中“备份导入失败”处理。旧数据恢复后如果内容不对，停止继续答题，先用另一份备份恢复。
